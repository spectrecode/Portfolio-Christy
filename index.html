Mis Habilidades

HTML

Lenguaje de etiquetas que sirven para desarrollar las páginas web; estas etiquetas son interpretadas por el navegador el cual hace visible en la pantalla lo que queremos lograr, formando desde títulos, trayendo imágenes etc. Es el esqueleto de toda página web.

CSS

Son reglas de estilos que definen cómo se vera tu página web, como ponerle color, ancho, alto, posición de imágenes y hasta animaciones.

Bootstrap

Es un Framework creado por Twitter, es decir, son etiquetas pre-diseñadas que nos permiten maquetar nuestras páginas web, haciendo que las mismas sean adaptables a distintos tipos de dispositivos.

JavaScript

Es un lenguaje de programacíon que se ejecuta en nuestras computadoras y/o dispositivos haciendo que nuestras páginas web cobren vida y animación con mejor cálidad y manipulación.

Jquery

Es una líbreria de Javascript, es decir, es un conjunto de códigos funcionales pre-definidos que simplifican y reducen un javascript puro

Sass

Simplifica y aligera el peso y extención de las reglas CSS, permitiendo repetición de estilos y/o reglas en el diseño de la página web.
